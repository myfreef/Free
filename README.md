# Bloodhero

Bloodhero is a modern, responsive blood donation web app concept built with Next.js.

## Implemented features
- Hero section with CTA and notice board
- Quick search by blood group, district, and area
- Donor registration form with full donor fields
- Donor list with filters and functional Call/WhatsApp actions
- User profile with flow system, privacy options, block/report actions
- Admin panel with role policy overview and donation statistics
- Language toggle (English/Bengali) and dark mode toggle
- Supabase client scaffold for backend integration

## Stack
- Next.js (App Router) + React + TypeScript
- Tailwind CSS
- Supabase JS client scaffold

## Run locally
```bash
npm install
npm run dev
```

> Note: package installation may be restricted in some managed environments.
